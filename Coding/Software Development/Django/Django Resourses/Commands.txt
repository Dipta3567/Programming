For Setup Globally: 

	1.  pip install Django

	2.  pip install pipenv

	3. pipenv --version


For Virtual Setup : 


	1. python -m venv .room1

	2. cd .room1

	3. Scripts\Activate

	4. django-admin startproject project_1  // for project

	5. cd project_1
	
	6. pip install Django

   	7. django-admin startapp first_app // for app

	8. python manage.py runserver

	9. Deactivate  ( FOR EXIT )
